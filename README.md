# 🏊 AquaDuo — Coach Natation

> Une méthode d'apprentissage de la natation assistée par intelligence artificielle.

## 🌐 Site en ligne
[aquaduo.app](https://aquaduo.app)

## 📁 Structure des fichiers

| Fichier | Description |
|---|---|
| `index.html` | Landing page |
| `swim_coach.html` | Application coach (conseils, séance, objectif, carnet) |
| `vercel.json` | Configuration Vercel |

## ✨ Fonctionnalités

- 🎯 **Conseils personnalisés** — exercices et mots exacts adaptés à l'âge et au niveau
- 📹 **Analyse vidéo** — Gemini AI analyse les défauts techniques de l'enfant
- 📅 **Séance du jour** — programme complet adapté au temps disponible
- 🎯 **Objectif du mois** — plan de progression sur 4 semaines
- 📓 **Carnet de suivi** — noter les progrès après chaque séance
- 🎙 **Dictée vocale** — décrire les difficultés à la voix
- 🌍 **Multilingue** — FR / NL / EN

## 🚀 Déploiement

Déployé automatiquement via [Vercel](https://vercel.com) à chaque push sur la branche `main`.
